## 🖥️ How to Install

### 🪟 Windows

1. Click the **Download for Windows** button above.
2. A file named **`mydocs_windows.exe`** will download.
3. Double-click the `.exe` file to run the installer.
4. If Windows shows a warning ("Unknown publisher"), click **More Info → Run Anyway**.
5. Follow the on-screen steps and click **Install**.
6. Once done, find **My Docs** in your Start Menu and launch it! 🎉

---

### 🍎 macOS

1. Click the **Download for macOS** button above.
2. A file named **`mydocs_mac_os.zip`** will download (check your `Downloads` folder).
3. **Double-click** the `.zip` file — it will automatically extract into a folder.
4. Open the extracted folder and drag **My Docs.app** into your **Applications** folder.
5. The first time you open it, macOS may say it "can't be opened because it's from an unidentified developer."

   **To fix this:**
   - Open **System Settings** → **Privacy & Security**.
   - Scroll down and click **"Open Anyway"** next to the My Docs message.
   - Click **Open** on the confirmation popup.

6. My Docs will now open normally every time! 🎉

> **💡 Tip for macOS:** When you open a PDF or document file in Finder, you can right-click it → **Open With → My Docs** to view it directly.

---

### 🐧 Linux

1. Click the **Download for Linux** button above.
2. A file named **`mydocs_linux.zip`** will download.
3. Right-click the `.zip` file → **Extract Here** (or use `unzip mydocs_linux.zip` in Terminal).
4. Open the extracted folder.
5. Find the file named **`mydocs`** (no extension) and double-click it to run.

   If it doesn't open, you may need to make it executable first:
   - Right-click the `mydocs` file → **Properties** → **Permissions** tab → Check **"Allow executing file as program"**.
   - Or in Terminal: `chmod +x mydocs && ./mydocs`

6. My Docs will launch! 🎉

## ❓ Frequently Asked Questions

**Q: Is My Docs free?**
> Yes! My Docs is completely free to download and use.

**Q: Do I need an internet connection?**
> No. My Docs works 100% offline. Your files never leave your computer.

**Q: Can I open a file by double-clicking it?**
> Yes! On Windows and macOS, you can set My Docs as the default app for PDFs and documents. Then just double-click any file to open it.

**Q: Why does Windows/macOS show a security warning?**
> My Docs is not signed with an expensive developer certificate yet. The app is completely safe. Follow the steps in the [installation guide](#️-how-to-install) above to bypass this warning.

**Q: Will My Docs edit my original files?**
> Only the Excel Editor modifies files (when you explicitly save). The PDF viewer, Word viewer, and CSV viewer are **read-only** — your originals are never changed.

**Q: What should I do if the app doesn't open?**
> - **Windows**: Right-click the `.exe` → Run as Administrator.
> - **macOS**: Go to System Settings → Privacy & Security → Click "Open Anyway".
> - **Linux**: Make the file executable first (`chmod +x mydocs`).
